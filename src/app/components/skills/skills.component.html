Habilidades

Las principales herramientas con las que trabajo son:

Tecnologías:

  • Java ☕ con Spring Boot para crear aplicaciones robustas y seguras.
  • Node.js ⚡ para integraciones rápidas y eficientes, como la integración de servicios de mensajería.
  • PostgreSQL y MySQL 🗄️ para la gestión eficiente de bases de datos relacionales.

  • Docker 🐋 y Docker Compose para la contenedorización y orquestación de microservicios.
  • Linux 🐧 como sistema operativo para el despliegue y administración de servidores y entornos de desarrollo.
  • Angular ⚛️ para el desarrollo de interfaces de usuario interactivas y responsivas.
